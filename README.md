Snakes N Ladders
================
A 10x10 board game

_*The game is created using [Ractive.js](http://www.ractivejs.org/).*_

Note : _*Supports maximum 3 players. A single player can also play.*_

##Browser support
Desktop
* IE >= 9
* Chrome (all versions)
* Firefix > 8
* Safari >= 5.0
* Opera >= 11.6

Mobile
* IOs >= 4.0
* Android >= 2.2
* Opera Mobile
